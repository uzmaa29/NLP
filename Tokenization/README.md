# Tokenization

A simple NLP notebook demonstrating text tokenization using Python and NLTK.

## overview

This notebook introduces tokenization, the process of breaking text into smaller units called tokens. It includes examples of word tokenization, sentence tokenization, and Tokem Frequency counter

## tech stack

* Python
* Jupyter Notebook
* NLTK

## note

This notebook was created for learning and practicing the fundamentals of Natural Language Processing (NLP).
