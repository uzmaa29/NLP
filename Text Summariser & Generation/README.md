

A simple NLP notebook that demonstrates text summarization using the Hugging Face Transformers library.

## tech stack

* python
* jupyter notebook
* hugging face transformers

## model

* `facebook/bart-large-cnn`

## what it does

* takes a paragraph as input
* generates a shorter summary using a pre-trained BART model

## how to run

1. install the required library:

   ```bash
   pip install transformers
   ```
2. open `notebook.ipynb`.
3. run the notebook cells
4. replace the sample text with your own if needed

## Note

this notebook was created for learning and experimenting with transformer-based text summarization.
